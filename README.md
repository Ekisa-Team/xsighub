## Iniciar proyectos Server y Mobile

1. Instalar `node_modules`.

   ```sh

   ```

2. Ejecutar aplicaciones.


- Mobile (Svelte native): [http://localhost:3001](http://localhost:3001)

  ```sh
  pnpm mobile:dev
  ```
